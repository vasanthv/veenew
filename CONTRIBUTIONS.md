## Veenew Contributions

Before opening a PR, please read the following guidelines.

Veenew is currently **not actively accepting contributions**. The primary purpose of making the Veenew source code available is to allow others to use the code for personal projects or to satisfy their curiosity, and to ensure code transparency, keeping the app honest about its actions.

If you’d like to support Veenew and its development, consider becoming a sponsor through [GitHub Sponsors](https://github.com/sponsors/vasanthv). You can also support the project by using Veenew, upgrading to the paid tier, or spreading the word about Veenew to your friends and family. Your support is greatly appreciated! 🙌
